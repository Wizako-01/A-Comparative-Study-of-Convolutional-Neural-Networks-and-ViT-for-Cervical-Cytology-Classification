# A-Comparative-Study-of-Convolutional-Neural-Networks-and-ViT-for-Cervical-Cytology-Classification
This project evaluates the performance of three distinct deep learning paradigms—ResNet50 (Residual CNN), EfficientNetB0 (Optimized CNN), and ViT-B-16 (Vision Transformer)—on the Herlev Pap Smear Dataset for automated cervical cancer screening.
